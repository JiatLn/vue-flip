# vue-fliplay

> Inspired by [react-flip-demo](https://github.com/MinJieLiu/react-flip-demo)

Flip is means **F**irst, **L**ast, **I**nvert, **P**lay.
This is a Vue 3 component, to make your list with animation if the data change.


## Play

```bash
npm install -g yarn

yarn install

yarn run play
```
